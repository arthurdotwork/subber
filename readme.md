# **Subber**

Subber is a golang CLI tool that interacts with a PubSub local emulator.

## **Installation**
`go get github.com/arthureichelberger/subber`

## **Commands**
### **Create a PubSub Topic.**

`subber createTopic`

### **Create a PubSub Subscription.**
`subber createSub`

### **List all topics.**
`subber listTopics`

### **List all subscriptions.**
`subber listSubs`
